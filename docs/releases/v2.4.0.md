# Titan v2.4.0 release notes

## Summary
Titan v2.4.0 focused on exporter reliability, retry-budget handling, and improved operational visibility ahead of the Beaconstone Q3 launch readiness review.

## Included changes
- Added Datadog monitor export support for tenant-facing diagnostics
- Improved retry budget handling in the API service
- Expanded release diagnostics to support launch validation workflows

## Why this mattered
This release improved operational confidence for teams monitoring retry-sensitive workflows and tenant export behavior during launch rehearsals.
